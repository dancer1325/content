# Ways to apply CSS to a document
* external stylesheet
* internal stylesheet
* inline styles

# Priority
* Cascade
  * at same style’s specificity → later styles override earlier styles
* Specificity
  * certain CSS selectors have higher priority (independently if another lower specificity style is declared afterwards)

# CSS property values
* Available types
  * keywords
  * numeric values
  * function
    * `calc(expressionToBeCalculated)`
* Shorthand options are available

# @rules
* `@import “pathToAnotherStyleSheet”`
  * If you don't add the top -> it doesn't work
* `@media`
  * criteria to apply certain style
